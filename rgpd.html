Politique de confidentialité UAS Manager

1. Responsable du traitement

CONTE Philippe · FRAjqq48j12nfy4c · 288, rue de la Concorde, 74300 THYEZ

2. Données collectées

UAS Manager collecte uniquement les données que vous saisissez vous-même : informations exploitant, données de vol, coordonnées GPS (avec votre autorisation), photos de missions, informations clients et devis.

3. Stockage & sécurité

Toutes les données sont stockées localement sur votre appareil via l'API localStorage du navigateur. Aucune donnée n'est transmise à nos serveurs. Les clés API (CheckWX, OpenWeatherMap) sont stockées de façon chiffrée localement.

4. Services tiers

L'application utilise des APIs tierces :
• CheckWX (checkwx.com) : transmission du code ICAO de la station
• OpenWeatherMap (openweathermap.org) : transmission de vos coordonnées GPS
• adsb.lol / airplanes.live : transmission de vos coordonnées GPS et du rayon de recherche
• IGN Géoportail : accès aux tuiles cartographiques publiques
• OpenStreetMap : accès aux tuiles cartographiques publiques

5. Vos droits (RGPD)

Conformément au RGPD (Règlement UE 2016/679), vous disposez des droits suivants :
• Accès : export JSON de toutes vos données
• Rectification : modification via l'interface
• Effacement : suppression via le module Config → Réinitialisation
• Portabilité : export JSON et CSV disponibles
• Opposition : désactivation des APIs dans Config → 🔑 API

6. Conservation

Les données sont conservées tant que vous utilisez l'application. La suppression du cache du navigateur ou la réinitialisation de l'application supprime définitivement toutes les données.

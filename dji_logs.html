Exporter depuis DJI Fly sur iPhone
1
Ouvrir l'app Fichiers
Appuie sur "Sur mon iPhone" dans le panneau de gauche de l'app Fichiers iOS.
2
Naviguer vers DJI Fly
Sur mon iPhone › DJI Fly › FlightRecords
3
Sélectionner le fichier .txt
Nom du fichier :
DJIFlightRecord_20260502_[143012].txt
4
Partager vers UAS Manager
Appui long → Partager → UAS Manager.
Ou revenir ici et sélectionner le fichier via le bouton "Sélectionner".
5
Alternative CSV (plus simple)
Dans DJI Fly → Profil → Mes vols → sélectionner → Partager → export CSV. Ce fichier est directement lisible, aucun déchiffrement requis.
Fichiers .txt DJI chiffrés
⚠Les fichiers .txt DJI Fly sont chiffrés par DJI. Cette app extrait les métadonnées de l'en-tête (date, appareil) sans déchiffrement. Pour les données complètes, préférer l'export CSV ou le service PhantomHelp.
